# SiteCraft

Premium websites for service businesses. Built with React, Vite, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

## Content

All site copy, images, and data lives in:
- `src/content.js` — all text, CTAs, pricing, FAQs etc.
- `src/examples.js` — example site cards with URLs
